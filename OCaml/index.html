Language Evaluation: OCaml
To Compare the Experience of Adopting Different Programming Languages
General Intent/Goals Functional + Imperative Hybrid with OOP
Competitors Haskell, F#, SML
Issues & Constraints Sub-par standard library/Standard library fragmentation (3 alternatives ones!); No type classes/function overloads; No multi-core support
Ease of Use
Complexity Great type inference; Modules as functions ("functors"); Currying by default; Structural typing
Learning Curve Medium
Install Process Install from OPAM or your package manager
External Library Install Process OPAM, some libraries available through other package managers
Learning Tools Official Tutorial
Active Beginner Community
Multiple Books Targeted to Beginners
Old-style Newsgroup
Multiple (Confusing) Newsgroups
StackOverflow Group
